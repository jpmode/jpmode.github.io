Recent Work

Content-Based Recommender System

Developed a recommender system that provided personalized book recommendations by analyzing user input on book titles and ratings. Extracted the Goodreads dataset from Kaggle and used FuzzyWuzzy for flexible title matching. Integrated weighted genres into cosine similarity calculations to prioritize user-preferred book genres.

Tools used: Python (Pandas, NumPy, FuzzyWuzzy, Scikit-Learn)

ElGamal Encryption Project

Implemented the ElGamal encryption algorithm from scratch for a class project. Users can input plaintext for encryption and decryption, supporting 256-bit encryption. The project visually demonstrates encryption changes with different bit sizes while maintaining successful decryption of the text.

Tools used: Python, SymPy (for mathematical operations)
